Projets réalisés durant le BTS

Voici les différents projets que le BTS SIO m'a permis de faire afin de m'inculquer différentes notions cruciales pour aborder le monde professionnel

Site Web Nolark

ce projet m'a permis de développer mes connaissances en HTML, Css et Javascript, savoir comment vréeer le squelette d'une page web avec HTML, la personnaliser avec le Css et vraiment lui donner vie avec du Javascript pout des personnalisations encore plus puissantes. dans ce projet j'ai pu utiliser des Api distants, créer un formulaire, gérer l'affichage de la page en mode impressions.

Portfolio d'Amélia Pilou

Ce projet m'a permis de savoir les bases de la gestion de versions avec git en local et à distance via un repository en ligne sur github; les notions de pull, commit, push ou encore merge qui étaient des notions inconnues pour moi m'est maintenant familière et ce projet m’a permis d'avoir les connaissances pour appliquer la gestion de versions comme une habitude dans mes projets futurs.

Étudiant de BAC général

Étudiant de BAC général, j'ai pu choisir la spécialité numérique science de l'informatique pour développer et avancer dans le monde du développement ainsi que du réseau, cette option m'a permis de créer de solide base me permettant d'aborder sereinement le BTS soi. Ayant choisi également l'option anglais monde contemporain, elle m'a permis de développer des capacités de langue anglaise et m'ouvre des possibilité internationale dans le futur.

Le BTS SIO

Passionné d'informatique depuis mon plus jeune âge, j'aime apprendre à faire de nouvelles choses avec mon ordinateur, mon smartphone ou ma tablette. Un Geek ? Oui assurément ! Mais j'aimerais maintenant passer au niveau au-dessus et devenir un vraie Nerd, dans le bon sens du terme ! Après mon bac, le BTS SIO s'est présenté comme une évidence. Avec 60% d'Informatique "pure", 13% d'Économie, management et droit de l'informatique, 9% d'Anglais, 9% de culture générale et 9% de Maths, je savais que cette formation allait me mettre le pied à l'étrier pour poursuivre ensuite dans cette voie et devenir informaticien!
